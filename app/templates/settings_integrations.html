Conexión WHM y Google Drive

Pasos recomendados

  1. En el servidor WHM: permita SSH solo desde la IP de este VPS de backup.
  2. En la carpeta del stack: secrets/whm_rsa (clave privada; la pública en authorized_keys del WHM).
  3. Volumen rclone_config: mismo nombre de remote que escriba aquí (p. ej. gdrive tras rclone config).
  4. Rellene los campos y pulse Guardar. Detalle y ejemplos: Instructivo completo.

Los valores guardados aquí sustituyen a los del .env cuando no están vacíos. La clave SSH debe existir en el contenedor (p. ej. montaje /secrets/whm_rsa). El token WHM API es opcional.

Token WHM API: deje el campo vacío para no modificar el valor ya almacenado; escriba un valor nuevo solo si desea actualizarlo.
